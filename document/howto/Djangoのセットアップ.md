## 環境

```
Python 3.8.5
```

## 手順
- venv内部での作業
### pipのアップグレード
`$ python -m pip install --upgrade pip`
### Djangoのインストール
`$ python -m pip install Django`
### Djangoプロジェクトの作成
`$ django-admin startproject conndot`
